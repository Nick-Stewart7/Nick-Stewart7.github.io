Tic Tac Toe

What is it?

The classic game of tic tac toe as a web application. The game supports 2 players and multiple spectators. You can log in to have your lifetime score tracked with a persisting postgreSQL database. The website was built using a Flask framework. Requests are handled by the Requests python library. Finally, React components were used to build the board and it's ruleset

What I learned

This was the first website I ever built and deployed. This process taught me there is so much more to building applications than functions and HTML. This project exposed me to the Flask framework, Javascript, React hooks, the Python Request library and real-world software design practices like flow control. I believe this project where my journey has really started because it put the entire computer industry in perspective for me.
